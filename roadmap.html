Section 01

The AI Engineering Landscape

You don't need a PhD in Mathematics or PyTorch training experience to build AI applications today. The modern AI Engineer glues together foundational models using APIs, Vector Databases, and Agentic logic.

Key Insight: AI Engineering is primarily a software engineering discipline with a layer of ML/AI understanding on top. If you can write Python and call REST APIs, you already have 60% of what you need.

Section 05

Learning Journey — All Three Levels

A structured, opinionated path from zero to production-ready AI systems.

1
Level 1 Python & API Foundations
Python is the lingua franca of AI. Master async I/O for non-blocking LLM calls, Pydantic for structured outputs, and FastAPI for serving your backend.
Python 3.12 Async / Await Pydantic v2 FastAPI REST APIs
2
Level 1 Prompt Engineering
Programming with natural language. Move beyond simple chat to crafting rigorous, structured system prompts with Chain-of-Thought reasoning and few-shot examples.
Chain of Thought Few-Shot XML Structuring System Prompts Temperature / Top-P
3
Level 1 Basic RAG Architecture
Connect models to your enterprise data. Learn embeddings, vector databases, and chunking strategies so LLMs read your docs instead of hallucinating.
Embeddings Pinecone / Qdrant pgvector Semantic Chunking Cosine Similarity
4
Level 2 Agentic Frameworks
Build systems that can act autonomously. Use LangGraph for stateful graphs, implement tool calling, and design human-in-the-loop approval workflows.
LangGraph CrewAI Tool Calling Memory & State HITL Patterns
5
Level 2 Model Context Protocol (MCP)
The USB-C of AI — a universal protocol to securely connect LLMs to your local environment, databases, and APIs. Build your own MCP servers in Python.
MCP Servers Secure Data Access stdio / SSE Transport Tool Registration
6
Level 2 Evaluation Pipelines
Data-driven AI Ops — build golden datasets, run LLM-as-a-judge scoring, and integrate evaluation into CI/CD so regressions never ship.
RAGAS Golden Datasets LLM-as-Judge CI/CD Integration Faithfulness Score
7
Level 3 Production LLMOps
Ship AI to production with confidence. Set up observability with LangSmith/Arize, implement cost controls, rate limiting, and automated prompt version management.
LangSmith Arize Phoenix Token Budgets Rate Limiting Prompt Versioning
8
Capstone Build a Portfolio AI Project
Apply everything in one end-to-end project: a RAG-powered agent with MCP tools, evaluated via RAGAS, deployed on a FastAPI backend with LangSmith observability.
Full-Stack RAG Agentic Loop MCP Integration Eval Dashboard GitHub Portfolio

Section 07

Level Deep-Dives

01

Core Fundamentals (Month 1–2)

Build the daily tools every AI Engineer uses — Python, Prompt Eng, and basic RAG.

Python & API Integrations

The Lingua Franca of AI: The entire ecosystem — LangChain, LlamaIndex, PyTorch, OpenAI SDK — is built on Python.

  • Async I/O: Critical for calling LLM APIs without blocking.
  • Pydantic: Structure and validate LLM outputs (Structured Outputs).
  • FastAPI: The standard for serving your AI backend.
asyncio pydantic fastapi httpx

Prompt Engineering

Programming with Natural Language: Crafting rigorous system prompts that produce consistent, structured responses.

  • Chain of Thought: Forcing reasoning before answering.
  • Few-Shot: Strict output formatting via examples.
  • XML Structuring: Tags like <context> to separate data.
CoT Few-Shot ReAct Self-Consistency

Basic RAG Architecture

Connecting Models to Data: Teaching LLMs to read your enterprise data instead of hallucinating.

  • Embeddings: Converting text into number arrays.
  • Vector DBs: Pinecone, Qdrant, pgvector.
  • Chunking: Splitting PDFs into meaningful paragraphs.
text-embedding-3 Qdrant pgvector
02

Advanced AI Engineering (Month 3–4)

Build systems that act autonomously and don't break in production.

Agentic Frameworks

Autonomous Workflows: LangGraph and CrewAI for intelligent loops with tool use and stateful memory.

  • Tool Calling: LLMs execute Python functions.
  • Stateful Graphs: Memory across multi-step tasks.
  • HITL: Pausing for user approval on sensitive actions.
LangGraph CrewAI AutoGen

Model Context Protocol

The USB-C of AI: A universal protocol to securely connect LLMs to your local environment and APIs.

  • Server Building: Custom Python MCP servers.
  • Secure Data Access: Granting Claude access to databases.
  • Standardisation: Away from custom tool-calling hacks.
@modelcontextprotocol stdio SSE

Evaluation Pipelines

Data-Driven AI Ops: RAGAS to mathematically score AI outputs. Never ship a prompt change blind.

  • Golden Datasets: 100+ Q&A pairs to benchmark against.
  • LLM-as-Judge: GPT-4 grades GPT-3.5 on Faithfulness.
  • CI/CD Integration: Fail PRs if accuracy degrades.
RAGAS DeepEval Arize
03

Production & Specialisation (Month 5–6)

Ship to production with observability, security, and FinOps in place.

LLMOps

Observability, cost management, and model versioning for AI systems in production at scale.

LangSmith Weights & Biases Helicone Token Budgets

AI Security

Defend against prompt injection, jailbreaks, and data exfiltration. Essential for enterprise AI deployments.

Prompt Injection Guardrails AI OWASP LLM Top 10 Garak

Open-Source Tools & FinOps

Run open-source models locally with Ollama, manage costs with token budgeting, and pick the right model for every task.

Ollama LM Studio vLLM Cost Modelling
