Module 19

Advanced AI Observability

Tracing multi-step agent executions and managing token usage across distributed systems.

1. OpenTelemetry for AI

OpenTelemetry (OTel) is the open standard for observability. AI applications map perfectly to it.

Traces & Spans

A Trace represents a full user request. A Span represents a single step (e.g. calling an LLM API, querying a Vector DB).

Metrics & Logs

Tracking Token Usage (input/output), Latency (Time to First Token), and logging exact prompts and responses.

2. Major Platforms

  • LangSmith: Native to LangChain. Incredible for tracing complex LangGraph agent executions.
  • Langfuse: Open-source observability, great for prompt management and cost tracking.
  • Arize Phoenix: Focuses heavily on RAG evaluation and embedding visualizations.
  • Helicone: Focuses on API proxying, caching, and rate limiting.
