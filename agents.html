Module 7

Agentic AI Systems

Go beyond simple chat. Build autonomous AI agents that reason, plan, execute tools, and iteratively solve complex multi-step problems.

1. What is an AI Agent?

An AI Agent is an LLM equipped with a system prompt that encourages reasoning, paired with a set of tools (functions) it can execute. Unlike standard LLMs which just output text, an agent outputs actions, observes the result, and loops until the goal is met.

🎯 Problem it solves

LLMs cannot act on the world (run code, fetch weather, send emails). Agents bridge the gap between "thinking" and "doing".

2. Real World Analogies

Kitchen Analogy

An LLM is a cookbook. An Agent is a Chef. The Chef reads the recipe (planning), grabs pans (tools), cuts veggies (action), tastes the soup (observation), and adds salt if needed (reflection).

DevOps Analogy

An Agent is like an autonomous CI/CD runner. If a build fails, it reads the error logs (observation), rewrites the broken dependency (action), and retries the build (looping).

4. Step-by-Step Tool Calling Workflow

When you ask an agent: "What is the weather in Tokyo?"

  1. System Prompt: LLM is given a list of available tools (e.g., get_weather(location)).
  2. Thought: The LLM thinks: "I need to use the get_weather tool for Tokyo."
  3. Action (Tool Call): The LLM outputs JSON: {"name": "get_weather", "arguments": {"location": "Tokyo"}}
  4. Execution: Your backend runs the actual API request, gets: {"temp": "22C", "condition": "Sunny"}
  5. Observation: The result is appended back to the LLM's context as a Tool Response message.
  6. Final Answer: The LLM generates: "It is currently 22Â°C and sunny in Tokyo!"

6. Common Mistakes

X Infinite Loops

Agent keeps calling the same failing tool because it doesn't understand the error.

OK Fix: Implement a max_iterations cap and force a specific failure message.

X Tool Overload

Giving one agent 50 tools confuses the LLM and degrades prompt context.

OK Fix: Use a Multi-Agent architecture. One Supervisor routes to specialized sub-agents with 3-5 tools each.
