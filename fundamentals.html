AI & LLM Fundamentals

Understand the core building blocks of Large Language Models: Tokens, Transformers, and Parameters.

1. What is an LLM?

A Large Language Model (LLM) is a type of neural network (specifically, a Transformer) trained on massive amounts of text data. Its core function is surprisingly simple: it predicts the next most likely word (or token) based on the context of the words that came before it.

2. Real World Analogy

Smartphone Autocomplete

An LLM is just a highly advanced version of your phone's autocomplete. When you type "I am going to the...", your phone predicts "store". An LLM does this across paragraphs and code blocks with vast logical reasoning.

4. Key Parameters

Temperature (0.0 to 1.0)

Controls randomness. A temperature of 0.0 makes the model deterministic (always picks the most likely next word). High temperature makes it creative and varied.

Context Window

The maximum number of tokens the model can "remember" in a single prompt. If a model has a 128k context window, it can read a ~300-page book in one go.

5. Interview Questions

Beginner

Q: Why does the model output different answers for the same prompt?

A: Because the Temperature is set > 0.0, introducing probability sampling into the next-token prediction.
