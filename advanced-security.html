Module 21

Advanced AI Security

Protecting agentic systems and production LLMs from sophisticated attacks.

1. Advanced Attack Vectors

Jailbreaking & Prompt Leakage

Bypassing safety filters to extract the hidden system prompt or force the model to output harmful content.

Data Exfiltration

Tricking an agent into reading a private database and then sending that data to an attacker-controlled external API.

Model Theft & Poisoning

Theft: Querying an API millions of times to train a clone model. Poisoning: Corrupting the training data so the model behaves maliciously.

2. Defense Layers

  • Input/Output Guardrails: Using tools like Nvidia NeMo Guardrails or LlamaGuard to scan prompts and outputs for PII or malicious intent before the LLM sees it.
  • Sandboxing: Running code-generating agents in secure Docker containers or WebAssembly so they can't access the host system.
  • Least Privilege Agents: If an agent only needs to read data, don't give its database tool WRITE permissions.
  • Human Approval Layers (HITL): Hard-coding interrupts into LangGraph so an agent cannot execute API calls without human consent.
