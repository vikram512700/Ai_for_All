Module 14 & 15

Open Source AI & FinOps

Learn how to run models locally to secure data and how to calculate and optimize the expensive API costs of cloud LLMs.

1. Open Source Local AI

Not everything needs to hit an API. By using open-source tools like Ollama or vLLM, you can run powerful models (like Llama-3 or Mistral) entirely on your local machine or private cloud infrastructure. This ensures zero data leakage and free inference.

Ollama

The easiest way to run local LLMs on your Mac/Windows PC. It runs in the terminal and provides a local REST API that perfectly mimics the OpenAI API format.

vLLM

An enterprise-grade, high-throughput LLM serving engine. It uses techniques like PagedAttention to maximize GPU memory utilization for production workloads.

2. AI FinOps (Cost Optimization)

API costs can spiral out of control. Calculating tokens effectively is a mandatory skill for AI Engineers.

3. Dynamic Model Routing

Don't use GPT-4/Claude 3.5 Sonnet for everything.

  1. Simple Tasks: (e.g., formatting JSON, extracting an email address). Route these to gpt-4o-mini or local Llama-3-8B. Costs $0.15 / 1M tokens.
  2. Complex Reasoning: (e.g., Agent planning, coding, multi-step math). Route to gpt-4o. Costs $15.00 / 1M tokens (100x more expensive!).
